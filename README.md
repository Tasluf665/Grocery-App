# 🛒 Nectar - Smart Grocery Shopping App

[![Download on Google Play](https://img.shields.io/badge/Download%20on-Google%20Play-blue?style=for-the-badge&logo=google-play)](https://play.google.com/store/apps/details?id=com.tasluf.groceryapp&pcampaignid=web_share)
[![Watch the Promo Video](https://img.shields.io/badge/Watch%20Promo%20Video-YouTube-red?style=for-the-badge&logo=youtube)](https://youtu.be/sNSF_BiEfO8?si=36aXZR7W5yiTBDcc)

Nectar is a **React Native** grocery shopping app that provides a seamless and efficient shopping experience. Built with **Expo, JavaScript, and Firebase**, it ensures fast performance, real-time order tracking, and a user-friendly interface.

## 🚀 Features  
✅ Browse & search for groceries easily 🥦🥛  
✅ Get exclusive deals & discounts 💰  
✅ Secure and fast checkout 💳  
✅ Real-time order tracking 🚚  
✅ User-friendly interface 📱  

## 🛠️ Tech Stack  
- **Frontend:** React Native (Expo)  
- **Backend:** Firebase (Authentication, Firestore, Storage)  
- **State Management:** React Context API

## 📥 Installation  
Clone the repository and install dependencies:  
```sh
git clone https://github.com/yourusername/nectar.git
cd nectar
npm install
```

```sh
npx expo start
```
